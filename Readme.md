```text

```



Each folder represents an independent Angular application.

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/angular-projects.git
cd angular-projects
```

---

### 2. Install dependencies (per project)

```bash
cd project001
npm install
```

---

### 3. Run the development server

```bash
ng serve
```

Open your browser at:

```text
http://localhost:4200
```


```text

```
